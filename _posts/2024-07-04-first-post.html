Self-written functions

In this analysis I used the following self-written functions:

Function for computing sample autocorrelation and partial sample autocorrelation:

def scf(y_t, h):
    T=len(y_t)
    y_bar=np.average(y_t)
    dif=y_t-y_bar
    gamma_not=np.sum(dif**2)/T
    gamma_h=np.zeros(h+1)  
    gamma_h[0]=gamma_not
    for i in range(1,h+1):
        gamma_h[i]=np.sum(dif[i:]*dif[:-i])/(T-i)
    rho_h=gamma_h/gamma_not
    phi_hh=np.zeros(h)
    phi_hh[0]=rho_h[1]
    for i in range(2,h+1):
        D= linalg.toeplitz(rho_h[:i])
        D_phi = linalg.toeplitz(rho_h[:i])
        D_phi[:,-1]=rho_h[1:i+1]
        phi_hh[i-1]=np.linalg.det(D_phi)/np.linalg.det(D)
    rho_h=rho_h[1:]
    return rho_h, phi_hh

Function that estimates AR(p) process using LS method

def est_arp(data, p, c, d): #p - order of AR process, d- dummy
    T = len(data) - p
    if (c == 1):
       X = np.zeros(shape=(T, p+1))
       X[:,0] = np.array(np.ones(T))
       for ii in range(1, p+1):
           X[:,ii] = data[p-ii:-ii]
    else:
        X = np.zeros(shape=(T, p))
        for ii in range(1, p+1):
            X[:,ii-1] = data[p-ii:-ii]
    if (d == 1):
        X=np.column_stack((X, dummy))
    res_arp = sm.OLS(data[p:], X).fit()
    alpha_est = res_arp.params
    y_hat = res_arp.fittedvalues
    res = res_arp.resid
    sig_est = res_arp.cov_params()
    return alpha_est, sig_est, res, y_hat, res_arp

Fuction that calculates three information criteria: Akaike Information Criterion, Hannan-Quinn Information Criterion, Schwarz or Bayesian Information Criterion. You can find the links to the original article below:

def info_crit(Y, maxLag):
    AIC = np.zeros(maxLag)
    HQ = np.zeros(maxLag)
    SC = np.zeros(maxLag)
    T = len(Y) - maxLag
    for p in range(1, maxLag + 1):
        y = Y[maxLag - p:]
        _, _, res, _, _ = est_arp(y, p, 1, 0)
        rss = np.mean(res**2)
        AIC[p-1] = np.log(rss) + (p + 1) * 2/T
        HQ[p-1] = np.log(rss) + (p + 1)* 2 * (np.log(np.log(T)))/T
        SC[p-1] = np.log(rss) + (p + 1)* np.log(T)/T
        
    val = np.array([AIC, SC, HQ])
    val1 = np.array([AIC, SC, HQ]).T
    infoC = np.amin(val, axis=1)
    lagOrder = np.argmin(val, axis=1)
    return infoC, lagOrder, val1

Function that de-means, standardizing function

def stand(res):
    T=len(res)
    mean_res=np.mean(res)
    sigma_tilde=np.sum((res-mean_res)**2)/T
    stand_res=(res-mean_res)/np.sqrt(sigma_tilde)
    return stand_res

The annual growth rate in \(\%\) is \(100 × ∆_{4} y_{t}\), where \(∆_{s} y_{t} ≔ y_{t} − y_{t−s}\) .

\[ γ_{h} ≔ \text{Cov}(y_{t}, y_{t-h}) = E[ (y_{t}-μ)(y_{t−h}-μ)] \]

Where \(μ ≔ \text{E}[y_{t}]\) for all \(t ∈ ℤ \).

The Autocorrelation Function (ACF) at lag \(h\) is computed as follows:

\[ ρ_{h} = Corr(y_{t}, y_{t-h}) = \frac{γ_{h}}{γ_{0}} \]

The Partial Autocorrelation Function (PACF) at lag \(h\) is computed as follows:

\[ φ_{hh} ≔ \text{Corr}(y_{t}, y_{t-h} ∣ y_{t − 1}, … , y_{t − h + 1}) \]

Remove seasonality with Dummy variables

The next step is that I am removing seasonality with a regression model. So, first I convert my seasonal dummies to time-series format and regress unemployment rate this dummies as follows:

\[ unemp_{t}=X\beta+\varepsilon_{t} \] where \(X\) data matrix includes dummy variables. One more important thing is I have to remove constant since I do not want to be caught dummy variable trap.

# Convert dummy dataframe to a time series object
